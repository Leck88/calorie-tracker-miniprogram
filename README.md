# 吃喝卡路里 - 微信小程序

一个简单实用的卡路里记录小程序，帮助你管理日常饮食，记录卡路里摄入，支持减脂餐导购和外卖推荐。

## 功能特点

### 📊 今日概览
- 直观展示今日卡路里摄入
- 热量目标进度可视化
- 蛋白质、碳水、脂肪摄入比例
- 快捷入口：减脂餐导购、外卖推荐

### 🍽️ 食品库管理
- **180+种** 预置常见食品数据
- 支持自定义添加新食品
- 按10+分类筛选、搜索食品

### 📝 饮食记录（早中晚加餐）
- **按餐次分类**：早餐、午餐、晚餐、加餐
- 各餐次热量分布统计
- 快速添加饮食记录
- 按日期查看历史记录

### 🥗 减脂餐导购
- 低卡减脂食品推荐（≤150千卡/100g）
- 高蛋白低脂肪食品推荐
- 科学减脂餐搭配方案（早/午/晚/加餐）
- 负卡路里食物清单
- 热量预警提示

### 🛵 外卖推荐
- 主流外卖平台快捷入口
- 低卡外卖选择建议
- 外卖点餐小技巧
- 高热量替代方案

### 💡 智能搭配
- 根据今日摄入推荐搭配方案
- 营养均衡提示
- 热量缺口/盈余分析

### 📈 数据统计
- 本周/本月热量趋势图
- 日均摄入分析
- 常吃食品排行榜

## 快速开始

### 开发环境
1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开微信开发者工具，点击「导入项目」
3. 选择本项目文件夹，填写 AppID（可使用测试号）

### 运行项目
1. 导入后点击「编译」
2. 即可在模拟器中预览效果
3. 使用微信扫描二维码可在真机预览

## 目录结构

```
├── app.js              # 小程序主入口（180+食品数据）
├── app.json            # 全局配置
├── app.wxss            # 全局样式
├── SPEC.md             # 规格文档
├── README.md           # 使用说明
└── pages/              # 页面目录
    ├── index/          # 首页（今日概览）
    ├── food/           # 食品库（180+食品）
    ├── add-food/       # 添加/编辑食品/记录
    ├── record/         # 饮食记录（早中晚加餐）
    ├── combine/        # 食品搭配
    ├── diet/          # 减脂餐导购
    ├── takeaway/      # 外卖推荐
    └── stats/          # 数据统计
```

## 数据存储

本小程序使用本地存储（wx.setStorage）保存数据：
- `foods`: 食品列表
- `records`: 饮食记录（含餐次类型）
- `settings`: 用户设置（每日目标等）

## 自定义配置

### 修改每日目标
在首页点击「目标」即可修改每日卡路里目标

### 添加新食品
1. 进入「食品库」页面
2. 点击右下角「+」按钮
3. 填写食品信息（名称、卡路里、营养成分等）

### 记录早中晚加餐
1. 点击首页「添加饮食」或底部「记录」tab
2. 选择餐次（早餐/午餐/晚餐/加餐）
3. 搜索并选择食品，输入克数
4. 点击「添加到今日饮食」

## 技术栈

- WXML + WXSS + JavaScript
